A Study of Enterprise Customer Self-Service

CHT 2013

With the evolution of Web-based self-service, business customers are able to use the information such as reporting a trouble ticket or checking their purchase status by themselves. That is to say, online self-service not only provides business customers with more powerful functionalities but also satisfies them by integrating with other channels which are service counters, telephoning, Websites, instant messages, mails, and emails. This paper shows the literature review of online self-service for enterprise customers, and then examines the Web self-service from Verizon, AT&T and Chunghwa Telecom. Therefore, with the comparison, we provide conclusive recommendations for improving self-service for the telecommunication industry.

A Study of Methods and Tools for Analyzing Security Vulnerabilities in Web Applications

NTU 2009

In this thesis, our objective is to evaluate the efficacy of existing verification cation tools. To this end, we build benchmark cases containing vulnerable code that causes security problems, such as cross-site scripting and SQL injection, but some benchmark cases do not have vulnerable code, to determine if a false positive occurs after scanning those code. Specifically, we use the developed benchmark cases to test four static analysis tools that generate reports of where the vulnerable code is, and to evaluate the performance of the tools statistically. As a result, we are able to know the advantages and disadvantages of each tool, so that suggest how to improve the efficacy in developing of the code-scanning tool.

e-GOVERNMENT – One Stop Shop

CCU 2007

e-Government in Taiwan provided 2,588 applications through online services in 2007, but we considered that the "document re-submitting" service is inconvenient to people. If we accidentally lost, for example, ID, re-submitting process is very troublesome since we have to take day off to apply for the new card. Even worse, people should go to many places if they lost more than one card. This porject carried out a new idea that people can easily re-submit documents via the Internet with a single window service system. The system involved in good communication mechanism with government agencies to achieve an efficient operation process.
